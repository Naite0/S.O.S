## 📌 Sobre o Projeto

O **S.O.S - Triagem Digital** é um sistema voltado para a **conscientização de sintomas** e **filtragem de urgências médicas**.

A aplicação permite que o usuário informe seus sintomas, intensidade e duração. Com base nesses dados, o sistema realiza uma triagem classificando o nível de risco:

🟢 Leve • 🟡 Médio • 🔴 Grave

Além disso, fornece orientações sobre quando procurar atendimento médico.

---

## 🎯 Objetivo

✔️ Orientar usuários sobre seus sintomas
✔️ Reduzir a superlotação em unidades de urgência
✔️ Promover educação em saúde
✔️ Melhorar o direcionamento no sistema público

---

## ⚠️ Problema

Muitos pacientes procuram **UPAs** mesmo com sintomas leves, como:

* Dor de cabeça
* Febre baixa
* Cansaço

Isso gera:

* 🚨 Superlotação
* ⏱️ Atraso em atendimentos graves
* 👩‍⚕️ Sobrecarga profissional

---

## 📊 Fundamentação

📍 No Ceará, em 2022:
➡️ **+691 mil atendimentos nas UPAs**
➡️ Grande parte de baixa complexidade

📢 Especialistas afirmam que muitos desses casos deveriam ser atendidos em postos de saúde (UBS).
